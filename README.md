Penguins Chat using React.JS, Firebase v9 with Google Login
=====================================

Live [demo](https://penguine-chat.netlify.app/)

Quick Start:
------------

- ``` git clone ```
- ``` cd react-firebase-chatApp ```
- Add your firebase configuration to firebase.js file on /src
- ``` npm install ```
- ``` npm start ```
